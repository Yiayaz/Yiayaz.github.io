CF710 F String Set Queries

题意

维护一个字符串集合，支持三种操作：

  1. 加字符串
  2. 删字符串
  3. 查询集合中的所有字符串在给出的模板串中出现的次数

本题强制在线，应该在每次输入后调用fflush(stdout)，你只有在输出上一个询问的答案后才能读入下一组询问。

思路

开两套AC自动机，一个插入一个删除，最后插入减去删除
